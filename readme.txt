=== Minimal Author Box Plugin ===
Requires at least: 3.1
Tested up to: 3.1
Stable tag: 0.1


Adds small box under blog posts with author's avatar, name, and description.

== Description ==

Adds small gray box under blog posts with author's avatar, name, and description.  The author's information is intentionally kept minimal and clean.
